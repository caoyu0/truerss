TrueRss is an open source feed reader. Via powerful plugin system, you might easy extend and extract information from any source, not only from atom or rss.

Usage

Just download jar file and run with one command

Plugins

Easy for create plugin for any source (email, twitter, your site)

Open source

Code of all plugins, and code of application is open
